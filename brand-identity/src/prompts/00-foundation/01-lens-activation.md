**When to Use:** At the very start of every new project session.
**Dependencies:** `src/resources/lenses/open-if-lens.md`.
**Output:** Confirmation only (no file generated).

## Prompt

Load `src/resources/lenses/open-if-lens.md` as analytical infrastructure.

The lens defines the Open If organising logic and evaluation discipline.
It is not the subject of the project.
It must not be summarised, extracted, positioned, or reframed.

Confirm:

- The lens will be used only to govern reasoning.
- The lens will not be treated as repository content.
- The lens will not become the entity being designed.
- The four-principle sequence (Invitation → Recognition → Coherence → Continuity) is now active as the evaluative framework for all downstream work.

Do not generate analysis.
Do not restate the lens.
Do not interpret it.
Return confirmation only.
